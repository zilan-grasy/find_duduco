# v1.0.0 (2026-05-08)

基线版本，对应 git 提交 `e162e75`（feat: 重构求解器并优化图像识别）。

## 功能
- **图像网格识别**：Canny边缘检测 + 投影峰 + Hough直线 → 网格线提取 → 切格子 → HSV取色 → K-means聚类 → 数字颜色矩阵
- **嘟嘟可求解算法**：约束传播（最后一格 / 鸽巢原理 / 包围排除）+ 双色mini回溯 + 主回溯
- **GUI界面**：Tkinter 主窗口，打开图像 / 截图识别 / 置顶
- **截图选择器**：全屏半透明遮罩，鼠标拖动框选
- `grid_recognizer_gui.py`：独立识别 GUI
- `solve_puzzle.py`：命令行求解器
