## v1.0.0 (2026-05-08)

基线版本，从零构建。

### 功能

**GUI 主界面**
- Tkinter 窗口，左上显示原图，左下显示识别出的颜色矩阵，右侧显示解题结果
- **打开图像**：选择本地 PNG/JPG 文件直接识别求解
- **截图识别**：全屏半透明遮罩 + 鼠标拖动框选棋盘区域 → 截取 → 识别 → 求解
- **置顶**：窗口始终保持在最前，方便对照游戏操作

**图像网格识别** — `GridImageRecognizer`
- Canny 边缘检测 → 行/列投影提取峰值 → Hough 直线补充 → 聚类合并去重 → 切出每个格子的子图
- 每个格子取中心 80% 区域（避开空隙边缘），HSV 均值取色
- 白/灰判定：饱和度 < 10 或 明度 > 245 且饱和度 < 25 视为空白格（0）
- K-means 聚类将相似颜色归为一类，Union-Find 合并距离 < 0.08 的近色簇

**嘟嘟可求解算法** — `DuducoPuzzleSolver`
- BFS 连通域：按颜色找出每个颜色的所有连通块
- 约束传播（不动点迭代）：
  - 规则① 最后一格：某行/列只剩 1 个合法位置 → 必放嘟嘟可，排除同行/列/相邻 8 格
  - 规则② 鸽巢原理：N 种颜色的候选位置仅跨 N 行/N 列 → 该 N 行/N 列内其他颜色排除
  - 规则③ 包围排除：某颜色的剩余合法格全落在同一格的 8 邻域内 → 该位置排除
- 双色 mini 回溯：约束传播卡住时取候选最少的 2 色穷举，跑完约束后交叉验证公共发现
- 主回溯：对剩余颜色递归穷举，保存/恢复状态
